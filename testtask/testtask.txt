Тестовое задание

1. При помощи wireshark снять дамп трафика, желательный размер дампа до 5 МБ:
1.1. HTTP, так чтобы в pcap файле была одна и только одна искомая сессия (любой HTTP сайт на выбор).
1.1. HTTPS, так чтобы в pcap файле была одна и только одна искомая сессия (любой HTTPS сайт на выбор).

2. Выполнить в ручном режиме с описанием действий (приложить сценарий ручного выполнения команд):
1) По протоколу FTP выгрузить какой-либо ZIP-архив.
2) Во время выгрузки снимать дамп трафика (*.pcap).
3) Из полученного дампа собрать выгруженый ZIP-архив
4) Проверить целостность архива, собраного из дампа pcap.

3. Написать скрипт автоматизации на Linux bash/shell.
1) По протоколу FTP выгрузить какой-либо ZIP-архив.
2) Во время выгрузки снимать дамп трафика (*.pcap).
3) Из полученного дампа собрать выгруженый ZIP-архив
4) Проверить целостность архива, собраного из дампа pcap.

Можно выбрать любой действующий FTP сервер.
Например:
ftp://ftp.alta.ru/packets/distr/
user=anonymous
